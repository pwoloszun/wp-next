1. make new npm project

$ npm init

2. create two TS modules

3. install TS & TS webpack loader - add to package.json:
  "devDependencies": {
    "awesome-typescript-loader": "^3.0.0-beta.17",
    "typescript": "2.0.10",
  },

4. install webpack - add to package.json:
  "devDependencies": {
    "webpack": "2.1.0-beta.25"
  },

5. config webpack

6. run webpack
  add build script

  "build": "webpack"

  $ npm run build

7. add script tag with built JS
